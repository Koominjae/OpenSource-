2. 가로줄 바꾸기

정답보드의 기본틀만 사용해 스도쿠를 생성한다면
(1, 2, 3), (4, 5, 6), (7, 8, 9) 끼리 모여있기 때문에 쉽게 풀리는 간단한 스도쿠만 생성 될것이다.

이를 방지하기 위해 각 가로줄 끼리 한번 더 섞어 줄 것이다.

* 주의) 	기본틀을 만드는 과정을 생각해보면 1, 4, 7번 기준 라인을 이용해 나머지 라인을 채웠다.
	이때 1 ~ 9번 라인들을 통째로 섞을 경우 같은 라인 또는 3x3 정사각형 안에 동일한 숫자가 중복해 들어갈 수 있다.
	그러므로 1, 2, 3번 라인 / 4, 5, 6번 라인 / 7, 8, 9번 라인끼리 섞어야 한다.

initialize_board_9x9를 함수를 보면 

	return [row1, row2, row3, row4, row5, row6, row7, row8, row9]

각 가로줄이 리스트의 인자로써 반환된다. 우선 가로줄을 3개씩 나누어 top, middle, bottom 변수에 저장한다.

그런 다음 random 모듈의 random.shuffle() 프로시저를 이용해
1, 2, 3번 라인 / 4, 5, 6번 라인 / 7, 8, 9번 라인끼리 무작위로 섞어준다.
그 다음 top, middle, bottom 에 저장된 리스트를 합쳐준다.

- 해당 작업을 수행하는 shuffle_ribbons 함수를 작성한다.
